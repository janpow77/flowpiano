# TROUBLESHOOTING.md — FlowPiano
Version 1.0

## Common Problems

### Second camera is unavailable
- Check whether the hardware combination supports MultiCam.
- Fall back to single-camera mode if necessary.

### No MIDI input detected
- Reconnect the keyboard.
- Verify that the device appears in system MIDI utilities.

### Virtual camera not visible
- Check extension installation state.
- Check signing / entitlement / approval status.

### No sound in conference app
- Verify `FlowPiano Virtual Mic` is selected.
- Check mic and piano gain.
- Check routing mode and feedback protection state.
